# Stroke — AInvest Design System

> 描边规范文档，定义分割线与组件描边两类场景的描边粗细与颜色使用规则。App 端与 Web 端在描边尺寸上规范不一致，但描边颜色规范两端一致。

---

## ⚠️ 重要使用注明（必须严格遵守）

**本规范中的全局描边样式不得覆盖（override）组件内固有的描边样式。**

具体说明：
- 标签（Tag）、Tab、按钮等控件类组件拥有自身固定的描边规格，以各组件规范文档中的定义为准；
- 本文档定义的全局描边变量仅适用于**分割线**以及**尚未在组件规范中明确描边值的元素**；
- 在任何实现中，若组件自身已定义描边，必须优先使用组件内固有样式，全局描边设置不得对其进行覆盖。

---

## 一、描边应用场景

描边主要应用于以下两类场景：

| 场景 | 说明 |
|------|------|
| **分割线** | 用于区隔内容区域、列表项、模块之间的视觉分隔 |
| **组件描边** | 标签、Tab、卡片、按钮等组件的边框描边（组件固有描边优先，见顶部注明） |

---

## 二、描边粗细规格

### 2.1 App 端（三档粗细）

| 等级 | 粗细 | 主要用途 |
|------|------|----------|
| **小号** | `0.5px` | 分割线 |
| **中号** | `1px` | 标签描边、Tab 描边、卡片描边、小号描边按钮 |
| **大号** | `1.5px` | Tab 多选中的选中态描边、中大号描边按钮 |

### 2.2 Web 端（两档粗细）

| 等级 | 粗细 | 主要用途 |
|------|------|----------|
| **小号** | `1px` | 分割线、标签、小号按钮 |
| **大号** | `2px` | 卡片描边 |

---

## 三、描边颜色规格

> Web 端与 App 端描边颜色规范一致，共分三档。Dark Mode 下将基准色由 `#000000` 替换为 `#FFFFFF`，透明度保持不变。

| 等级 | Light Mode | Dark Mode | 主要用途 |
|------|-----------|-----------|----------|
| **Level 1** | `rgba(0,0,0,0.20)` | `rgba(255,255,255,0.20)` | 列表表头默认分割线 |
| **Level 2** | `rgba(0,0,0,0.10)` | `rgba(255,255,255,0.10)` | 分割线、App 端 1px 卡片描边 |
| **Level 3** | `rgba(0,0,0,0.05)` | `rgba(255,255,255,0.05)` | Web 端 2px 卡片描边 |

---

## 四、粗细与颜色的搭配组合

### 4.1 通用搭配规则

| 平台 | 粗细 | 颜色等级 | 典型用途 |
|------|------|----------|----------|
| App | `0.5px` | Level 2 | 分割线 |
| App | `1px` | Level 2 | 标签描边、Tab 描边、卡片描边、小号描边按钮 |
| App | `1.5px` | Level 2 | Tab 选中态描边、中大号描边按钮 |
| Web | `1px` | Level 1 | 列表表头分割线 |
| Web | `1px` | Level 2 | 分割线、标签、小号按钮 |
| Web | `2px` | Level 3 | 卡片描边 |

### 4.2 股票列表场景规范

| 场景 | 平台 | 粗细 | 颜色等级 | 色值（Light） |
|------|------|------|----------|--------------|
| **股票列表表头分割线** | App | `0.5px` | Level 1 | `rgba(0,0,0,0.20)` |
| **股票列表表头分割线** | Web | `1px` | Level 1 | `rgba(0,0,0,0.20)` |
| **股票列表数据项分割线** | App | `0.5px` | Level 2 | `rgba(0,0,0,0.10)` |
| **股票列表数据项分割线** | Web | `1px` | Level 3 | `rgba(0,0,0,0.05)` |

## 五、禁止事项（Forbidden）

| ❌ 禁止 | ✅ 正确做法 |
|---------|-----------|
| 用全局描边样式覆盖按钮、Tab、标签等组件的固有描边 | 优先使用各组件规范文档中定义的描边值 |
| Web 端使用 `0.5px` 描边 | Web 端最小描边为 `1px` |
| App 端使用 `2px` 描边 | App 端最大描边为 `1.5px` |
| 使用规范外的自定义描边粗细（如 `3px`、`0.8px`） | 严格使用规范内定义的粗细值 |
| 使用规范外的自定义描边颜色 | 严格使用 Level 1 / Level 2 / Level 3 三档颜色 |
| Web 端与 App 端混用描边粗细规格 | 两端描边粗细独立，按各自规范执行 |
